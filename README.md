# Expense Tracker

A full-stack web application for recording, categorizing, and tracking daily expenses.  
This project demonstrates end-to-end development using a simple HTML/CSS/JavaScript frontend, a Spring Boot backend, and MongoDB for persistent data storage.

---

## Features

- Add, edit, delete, and view expenses  
- Categorize expenses (Food, Travel, Shopping, Bills, etc.)  
- REST API built with Spring Boot  
- MongoDB-powered persistence layer  
- Clean and responsive UI  
- Easy to set up and run locally

---

## Tech Stack

**Frontend:** HTML, CSS, JavaScript  
**Backend:** Java, Spring Boot, Maven  
**Database:** MongoDB
